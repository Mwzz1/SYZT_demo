# 《深渊征途》玩法介绍

## 游戏概述

《深渊征途》是一款回合制RPG刷宝游戏，你将扮演一位勇者，在幽暗森林、遗忘洞穴、恶魔遗迹、远古龙巢等地图中不断挑战怪物，击败Boss，获取装备，提升实力。游戏的核心循环是：
**战斗 → 获取经验/装备 → 强化角色 → 挑战更强敌人**。

---

## 一、战斗系统

### 基本战斗流程
- 战斗为**回合制**，玩家与敌人轮流行动

### 玩家可执行的行动

| 行动 | 效果 |
|------|------|
| ⚔️ **普通攻击** | 造成物理伤害，获得 **5 SP**（暴击获得 8 SP） |
| 🔥 **技能** | 消耗 SP 释放强力技能，有冷却时间 |
| 🛡️ **防御** | 减少受到的伤害，获得 **3 SP** |
| 🧪 **药水** | 使用背包中的药水恢复 HP 或 SP |
| 💼 **装备背包** | 查看和更换已获得的装备 |
| ⭐ **技能树** | 消耗技能点学习天赋 |
| 🧬 **职业** | 切换职业（战士/法师/刺客/圣骑士） |
| 💾 **存档管理** | 保存或读取游戏进度 |
| ➡️ **下一战** | 击败当前怪物后进入下一关 |

### SP（战技点）系统
- SP 是释放技能的关键资源，上限 50
- **普通攻击**：+5 SP（暴击 +8 SP）
- **防御**：+3 SP
- **技能**：消耗 SP，数值越高效果越强
- ⚠️ SP 不足时无法释放技能，需要用普攻或防御积攒

### 技能类型
- **通用技能**：强力斩击、火球术、闪电链、毒刃、冰霜新星
- **职业技能**（随职业切换）：
  - 战士 → 旋风斩
  - 法师 → 奥术爆破
  - 刺客 → 影袭
  - 圣骑士 → 圣光审判
- **特殊技能**：龙神吐息（需要龙神套装 3 件解锁）

## 二、角色成长系统

### 等级与经验
- 击败怪物获得经验值，经验满后自动升级
- 升级提升：最大 HP/SP、攻击/防御/速度，并获得 **1 技能点**

### 属性说明

| 属性 | 作用 |
|------|------|
| ❤️ HP | 生命值，归零则战斗失败 |
| ⚡ SP | 战技点，释放技能的消耗资源 |
| ⚔ ATK | 攻击力，影响造成伤害 |
| 🛡 DEF | 防御力，减少受到的伤害 |
| 💨 SPD | 速度，影响行动顺序 |
| 💥 CRIT | 暴击率，暴击时伤害 ×1.6 |
| ⭐ GS | 装备评分，综合装备品质的数值 |

### 职业系统
可在 **战士 / 法师 / 刺客 / 圣骑士** 之间切换，每个职业提供不同的属性加成和专属技能：

| 职业 | 加成 | 专属技能 |
|------|------|----------|
| 战士 | HP +30%，攻击 +15% | 旋风斩 |
| 法师 | SP +50%，技能伤害 +30% | 奥术爆破 |
| 刺客 | 速度 +25%，暴击 +20% | 影袭 |
| 圣骑士 | 防御 +25%，治疗增强 | 圣光审判 |

### 技能树（天赋）
消耗技能点学习，可多次升级：

| 天赋 | 效果（每级） | 解锁等级 |
|------|-------------|----------|
| 强健体魄 | HP +25 | Lv.1 |
| 武器大师 | 攻击 +5 | Lv.1 |
| 致命一击 | 暴击 +5% | Lv.5 |
| 火焰强化 | 技能伤害 +8 | Lv.5 |
| 连斩 | 普攻伤害 +7 | Lv.10 |
| 吸血 | 吸血 +3% | Lv.10 |
| 龙神斩 | 技能伤害 +20 | Lv.15 |
| 雷神降临 | 暴击 +8%，速度 +5 | Lv.15 |

## 三、装备系统

### 装备部位
- **武器**（⚔️）：主要增加攻击力
- **防具**（🛡️）：主要增加防御力
- **饰品**（💍）：主要增加暴击、速度等

### 装备品质（稀有度）
| 品质 | 颜色 | 词条数 |
|------|------|--------|
| 普通（Common） | 灰色 | 0 |
| 稀有（Rare） | 蓝色 | 1 |
| 史诗（Epic） | 紫色 | 2 |
| 传说（Legendary） | 橙色 | 3 |
| 神话（Mythic） | 红色 | 4 |

### 装备词条
装备可能附带额外属性：锋锐（攻击）、坚固（防御）、疾风（速度）、幸运（暴击）、吸血、燃烧概率、雷击概率等。

### 装备获取
- 击败怪物概率掉落
- Boss 必定掉落装备
- 商店购买装备箱

### 装备强化（🔨）
- 消耗**金币**和**强化石**提升装备等级（最高 +10）
- 每级强化提升主属性，同时增加装备评分

### 装备分解（♻️）
- 分解装备获得：金币、强化石、精华
- **重铸词条**：消耗精华重新随机装备的词条

### 套装系统
收集同系列装备（2件/3件）激活额外效果：

| 套装 | 2件效果 | 3件效果 |
|------|---------|---------|
| 🔥 火焰套装 | 攻击 +15% | 火焰技能伤害 +25%，燃烧概率提升 |
| 🐲 龙神套装 | 攻击 +20% | 暴击 +15%，解锁龙神吐息 |
| ✨ 创世套装 | 全属性 +20% | 攻击时概率召唤天罚 |

## 四、资源系统

| 资源 | 获取方式 | 用途 |
|------|----------|------|
| 💰 金币 | 击败怪物掉落 | 购买药水、强化石、装备箱，旅店恢复，装备强化 |
| 🔨 强化石 | Boss 必掉、精英概率、商店购买 | 装备强化 |
| 🧪 药水 | 怪物概率掉落、商店购买 | 战斗中恢复 HP/SP，复活 |
| 🌀 精华 | 分解史诗以上装备获得 | 重铸装备词条 |
| ⭐ 技能点 | 升级获得、Boss 击杀奖励 | 学习天赋 |

### 药水类型
- **小型生命药水**：恢复 35% 最大 HP
- **大型生命药水**：恢复 70% 最大 HP
- **战技药水**：恢复 45% 最大 SP
- **复活药水**：死亡时自动复活，恢复 50% HP

## 五、地图与关卡

### 地图区域
| 地图 | 关卡范围 |
|------|----------|
| 🌲 幽暗森林 | 第 1 - 10 关 |
| 🕳️ 遗忘洞穴 | 第 11 - 20 关 |
| 🏛️ 恶魔遗迹 | 第 21 - 30 关 |
| 🐲 远古龙巢 | 第 31 关起 |

### 关卡机制
- 每 10 关有一个 **Boss**（哥布林王、骷髅领主、恶魔领主、龙神）
- Boss 拥有**特殊机制**：
  - 龙神：每 3 回合释放龙息（造成最大生命 18% 伤害）
  - 骷髅领主：每 4 回合恢复 8% 生命
  - 低血量时进入狂暴，攻击力大幅提升
- 击败 Boss 额外奖励技能点，Boss 关卡有特殊视觉和音效反馈

### 怪物类型
| 类型 | 强度 | 掉落 |
|------|------|------|
| 普通怪 | 标准 | 概率掉落药水、装备 |
| 精英怪 | ×1.45 | 更高概率掉落，概率掉强化石 |
| Boss | 特殊 | 必掉装备 + 强化石 + 技能点 |

| 奖励类型 | 效果 |
|----------|------|
| 攻击祝福 | 攻击 +6 |
| 生命祝福 | HP +35 |
| 战意祝福 | SP +20 |
| 补给包 | 小型生命药水 +1，战技药水 +1 |
| 幸运祝福 | 暴击 +4% |
| 特殊祝福 | 炽焰祝福、雷霆祝福、猎杀祝福、贪婪祝福、鲜血祝福等 |

## 八、存档系统（💾）

- 支持**多存档槽**（最多 8 个）
- 快捷键：**Ctrl+S** 快速保存，**Ctrl+L** 打开读档界面
- 自动兼容旧版本存档

---

## 九、流派强度系统

游戏会根据你的装备、套装、祝福计算三个流派的强度分数：

| 流派 | 评分依据 |
|------|----------|
| 🔥 火焰流 | 火焰套装件数 + 燃烧概率 + 炽焰祝福 |
| ⚡ 雷电流 | 雷击概率 + 雷霆祝福 |
| 💥 暴击流 | 暴击率 + 猎杀祝福 + 刺客职业加成 |

---

## 十、战斗特效与视觉

- 技能释放伴有**投射物动画**（火球、闪电、龙息等）
- 暴击时特效增强（金色刀光、屏幕震动）
- Boss 战有特殊的攻击动画和光环效果
- 动态战场背景（根据地图变化）、雾气、月光等氛围元素

---

## 快速上手指南

1. **开局**：点击“普通攻击”击败第一个敌人
2. **积攒 SP**：用普攻或防御积累 SP，再释放强力技能
3. **更换装备**：掉落装备后比较评分，择优装备
4. **学习天赋**：升级后记得使用技能点
5. **挑战 Boss**：每 10 关有 Boss，建议备好药水
6. **收集套装**：优先凑齐 2 件套获得加成
7. **保存进度**：按 Ctrl+S 随时保存
